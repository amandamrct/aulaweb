A diferença entre a linguagem de programação Server Side e a Client Side é que, na primeira, o código é interpretado no servidor, que controla as informações enviadas ao usuário e processa o código para responder ao navegador; enquanto a segunda quem interpreta o código é o navegador do cliente, que é quem lida com a estrutura e a apresentação dos dados para o usuário. Exemplo de Server-side: PHP Exemplo de Client-side: Javascript
